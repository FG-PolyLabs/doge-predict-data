# doge-predict-data

Auto-generated JSON data files for the Doge Predict crypto betting analytics project.

**Do not manually edit these files.** They are committed by the [backend](https://github.com/FG-PolyLabs/doge-predict-backend) via scheduled jobs and API calls.

## Data Files

| File | Description |
|------|-------------|
| `tracked_tokens.json` | Crypto tokens being tracked |
| `price_snapshots.json` | Historical price data |
| `betting_markets.json` | Active betting markets |
| `market_snapshots.json` | Historical market odds |
| `strategies.json` | Defined betting strategies |
| `backtest_results.json` | Backtest execution results |

Part of the [FG-PolyLabs/doge-predict](https://github.com/FG-PolyLabs) four-repo project.
